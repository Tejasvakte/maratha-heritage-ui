# maratha-heritage-ui
# 🏰 Shivneri Fort – Premium Historical Website

A visually stunning, modern, and interactive website showcasing **Shivneri Fort**, the birthplace of **Chhatrapati Shivaji Maharaj**.

Designed with a focus on **premium UI, smooth animations, and storytelling**, this project transforms historical content into a cinematic web experience.

---

## ✨ Features

* 🎨 **Premium UI Design**

  * Clean, minimal, and modern layout
  * Glassmorphism cards with blur effects
  * Royal saffron & gold theme

* 🎬 **Smooth Animations**

  * Scroll-based reveal animations
  * Hover interactions (cards, buttons, images)
  * Parallax background effects

* 🖼️ **Cinematic Background**

  * Blurred image of Shivaji Maharaj
  * Dark overlay for readability

* 📱 **Fully Responsive**

  * Works on mobile, tablet, and desktop

* 🧭 **Smooth Navigation**

  * Sticky navbar
  * Section-based scrolling

---

## 📂 Sections Included

* Hero Section (Intro + Title)
* About Shivneri Fort
* Historical Importance
* Architecture Highlights
* Image Gallery
* Timeline of Shivaji Maharaj
* Visitor Information
* Footer
